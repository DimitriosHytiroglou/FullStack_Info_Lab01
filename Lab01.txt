1. Change Directory: cd <path>
2. Create a Directory: mkdir <directory>
3. Rename a Directory: mv <originalPathAndFile> <newPathAndFile>
4. List the contents of a directory: ls
5. Move a file to a new location: mv <originalPathAndFile> <newPath>
6. Copy a file to a new location: cp <originalPathAndFile> <newPathAndFile>
7. Search for a string in a file: grep "string" <Filepath>
8. Access the shell of a remote machine: ssh remote_username@remote_host 
9. Remove a directory folder: rm -r/-rf <directory>
